Analysis of mother & daughter yeasts

Description of the problem

Segmentation and analysis of mother and daughter yeasts.
The first channel can be used for the segmentation phase.
The second channel is stained in a way that reveals the presence of a protein in the membrane.

1. Links

  • Page on Redmine
  • GitHub repository

2. What about the project?

  • The goal is to analyse the intensity of the membrane of both the mother and the daughter.
  • The segmentation must be realized in the first channel. The extracted areas must be used in channel 2 to aggregate data.
  • The input is composed of an image on which the user will provide ROIs.
  • The first ROI is a polygon that will be used to extract the background.
  • The following ROIs are points ROIs (two points per ROI). A point in the mother, and another one in the daughter.
  • All measures have to be processed (mean, min, max, med, Q1 & Q3) and extracted as a CSV.
